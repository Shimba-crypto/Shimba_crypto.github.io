## ✨ Features
- **30 Curated Titles:** Hand-picked games across various genres (FPS, Sandbox, Racing, RPG).
- **Glassmorphic UI:** Modern, semi-transparent interface with smooth hover effects.
- **Theme Toggle:** Fully functional Dark/Light mode with persistent memory (LocalStorage).
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Smooth Animations:** Integrated with `Animate.css` for a dynamic user experience.
- **Verified Links:** Every link points to official stores (Steam, Epic, Google Play) to ensure user safety.

---

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables)
- **Framework:** Bootstrap 5.3
- **Animations:** Animate.css
- **Logic:** Vanilla JavaScript (ES6+)
- **Hosting:** GitHub Pages
